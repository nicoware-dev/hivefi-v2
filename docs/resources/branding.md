# HiveFi Brand Guidelines

## Brand Overview

HiveFi's brand identity represents the convergence of artificial intelligence and decentralized finance, emphasizing intelligence, automation, and connectivity.

## Logo

### Primary Logo
```
assets/logo/
├── hivefi-logo-full.svg    # Full logo with text
├── hivefi-logo-symbol.svg  # Symbol only
└── hivefi-logo-text.svg    # Text only
```

### Logo Variations
- Full Color (Default)
- Monochrome (Black)
- Monochrome (White)
- One Color (Brand Primary)

### Clear Space
Maintain minimum clear space equal to the height of the "H" in the logo on all sides.

### Minimum Size
- Digital: 100px width
- Print: 1 inch width

## Color Palette

### Primary Colors
```css
--brand-primary: #FF6B2C;    /* Vibrant Orange */
```

- Active voice
- Technical accuracy
- User-focused
